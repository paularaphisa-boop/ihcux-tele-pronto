Nomes: Julia Beatriz, Paula Raphisa, Ryan Pierre e Heitor Sousa.

Análise de Acessibilidade O design do aplicativo foi pensado para ajudar usuários que estejam passando mal, com visão turva ou dedos trêmulos. Por isso, utilizamos botões grandes, textos com boa legibilidade, ícones simples e poucas informações por tela. Também foram usadas cores com bom contraste e atalhos rápidos para facilitar o acesso às funções principais sem dificuldade.

Fluxo Crítico O usuário inicia a consulta de urgência pela tela inicial, clicando no botão “Consulta Agora”. Em seguida, ele passa pela triagem, seleciona os sintomas ou a região da dor e descreve o problema. Depois disso, o sistema encaminha o usuário para uma consulta online com um médico disponível.

Prevenção de Erros Para evitar que o usuário encerre a consulta sem querer, o aplicativo possui um botão de encerramento maior e separado dos outros controles. Além disso, aparece uma mensagem de confirmação perguntando se o usuário realmente deseja finalizar a consulta antes da ação ser concluída.
